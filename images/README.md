# Images

- `logo.svg` — recreated vector version of the restaurant logo (used as a fallback).
- **`logo.png`** *(recommended)* — drop your real logo PNG here with this exact name.
  The site automatically prefers `logo.png` if present and falls back to `logo.svg`.

To replace the logo:
1. Save your high-res logo as `images/logo.png`.
2. Refresh the site — no code changes needed.
